## 1. Single Company Funds (Standard Deals)

| Field                    | Value                                                                 |
|--------------------------|-----------------------------------------------------------------------|
| **Fund Name**            | [Company Name] Fund (varies per deal)                                 |
| **Structure**            | Limited Partnership pooling investor capital for single-company shares |
| **Strategy**             | Pre-IPO secondary market — acquire fully vested shares from employees |
| **Minimum Investment**   | $10,000                                                               |
| **Management Fee**       | N/A (one-time transaction fee instead)                                |
| **Transaction Fee**      | 3–5% of investment amount (one-time, at closing)                      |
| **Performance Fee**      | None (no carried interest or AUM fee post-transaction)                |
| **Seller Fee**           | 5% placement fee (deducted from seller proceeds)                      |
| **Liquidity**            | Illiquid — locked until IPO, acquisition, or secondary sale           |
| **Lock-up Period**       | 2–5 years expected                                                    |
| **Historical Returns**   | Range: -100% to +450% across 250+ companies (wide dispersion)        |
| **Dividends / Income**   | None expected — no dividends, payments, or cash flow                  |
| **Investor Eligibility** | Accredited investors only                                             |
| **Number of Funds**      | 2,200+ funds across ~500 companies; on 350+ cap tables                |
| **Total Investments**    | 51,000+ investments made on the platform                              |
| **Sharpe Ratio**         | Not publicly disclosed                                                |
| **Max Drawdown**         | Not applicable (illiquid, no mark-to-market)                          |
| **Volatility**           | Not applicable (illiquid, no mark-to-market)                          |

### Notes
- Investors own a pro-rata right to shares held by the fund, not direct shares.
- Share type: fully vested stock (not options or RSUs).
- Exit: fund distributes shares or cash directly to investors after any lock-up period.

## 2. Express Deals

| Field                    | Value                                                                 |
|--------------------------|-----------------------------------------------------------------------|
| **Fund Name**            | Express Deal (secondary transfer of existing fund interest)           |
| **Structure**            | Transfer of existing LP interest in a Single Company Fund             |
| **Closure Timeline**     | As fast as 3 days                                                     |
| **Minimum Investment**   | Varies (must purchase full sell order)                                 |
| **Fees**                 | Same fee structure as standard deals                                  |
| **Seller Requirements**  | Min 1-year holding; >2% ownership of fund shares; sale >2% threshold  |
| **Liquidity**            | Faster entry, but underlying shares remain illiquid until exit event   |

## 3. Multi-Company Funds

| Field                    | Value                                                                 |
|--------------------------|-----------------------------------------------------------------------|
| **Sub-type**             | Growth Opportunity Fund / Thematic Fund                               |
| **Structure**            | Diversified fund across multiple pre-IPO companies                    |
| **Strategy**             | Growth Opportunity: broad mid-to-late stage tech (15–25 companies)    |
|                          | Thematic: industry-focused (4–6 companies, e.g., AI/ML, Fintech, Future of Food) |
| **Minimum Investment**   | $50,000 (occasionally $20,000 slots available)                        |
| **Fund Life**            | 5–7 years expected                                                    |
| **Management Fee**       | Not publicly disclosed                                                |
| **Transaction Fee**      | Not publicly disclosed for multi-company funds                        |
| **Performance Fee**      | Not publicly disclosed                                                |
| **Historical Net IRR**   | 15–25% (historical range for curated funds)                           |
| **Liquidity**            | Illiquid — cash distributed as portfolio companies exit               |
| **Sharpe Ratio**         | Not publicly disclosed                                                |
| **Max Drawdown**         | Not applicable (illiquid, no mark-to-market)                          |

### Notes
- Growth Opportunity Fund provides broad exposure to private tech market.
- Thematic Funds concentrate in specific sectors.
- Past thematic fund themes: Future of Food, AI/Machine Learning, Fintech.

## 4. Direct Share Acquisitions

| Field                    | Value                                                                 |
|--------------------------|-----------------------------------------------------------------------|

| **Fees**                 | Not publicly disclosed (negotiated per deal)                          |
| **Liquidity**            | Illiquid — same constraints as above                                  |
| **Key Benefit**          | Direct negotiation on price/size; independent relationship with company |
